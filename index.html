Python: Qué es

  • Lenguaje interpretado
  • Tipos dinámicos / Tipos fuertes
  • Orientado a objetos
  • Multi-propósito
  • Pensado para ser rápido (de aprendrer, de usar, de entender)
  • Pensado para crear un código uniforme y limpio

Python: Algunas ventajas

  • Permite abordar problemas simples de forma simple y problemas complejos
    • Programación tipo script
    • Programación orientada a objetos con excepciones, etc
  • Librería estándard muy grande: "batteries included"
  • Muchas librerías y frameworks que lo extienden
    • Creación de entornos web
    • Administración de sistemas
    • Llibrerías científicas y geogràfiques
    • Programación en paralelo
  • Python une librerías de distintos lenguajes.
  • Tiene aplicaciones que permiten usarlo de forma interactiva ipython, SAGE

GDAL/OGR

Geospatial Data Abstraction Library
  • Modelo de datos abstracto para leer formatos GIS raster y vectorial
  • Más de 150 formatos soportados:
    • GeoTIFF
    • ENVI
    • Shapefile
    • PostGIS
    • GRIB
    • NetCDF
    • VAISALA IRIS
  • Usada por muchas aplicaciones y programada en C, tiene bindings en Python
  • Lectura/Escritura/Cambios de coordenadas/Operaciones GIS

Formato WKT de las proyecciones

'PROJCS["ETRS89 / UTM zone 31N", Nombre de la proyección (arbitratrio)

SPHEROID["GRS 1980",6378137,298.2572221010002, AUTHORITY["EPSG","7019"]], Definición del elipsoide
